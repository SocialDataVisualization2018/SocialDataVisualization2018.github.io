Project B: Explainer Page

Motivation

  • What is our dataset?

    • We used the dataset called 'NYPD Motor Vehicle Collisions' which describes the collision in NYC by using location and injury, each record represents a collision in NYC by city,borough, precinct and cross street,this data can be used by the public to see how dangerous/safe intersections are in NYC, it includes 1.26M rows and 29 columns. We used API to get the data instead of downloading csv file because of keeping data integrity.

  • Why did you choose this/these particular dataset(s)

    • At the first time we got this dataset we found that this dataset is easy to understand and interesting to explore,it has suitable amount of observation and attributes, the data Integrity is preferable to handle with, besides that, it has coordinate attributes so that we can use different type of graph(e.g GeoMap) to show the most meaningful imformation.

  • What was your goal for the end user's experience?

    • The goal of user's experience is that the use can operate on the graph, which include select the specific area of NYC to check its Motor Vehicle Collisions and they are also should be shown the dynamic line chart to show the changing curve with time passing.

Basic stats

  • Write about your choices in data cleaning and preprocessing

    • 1) Since the data intergrity is preferable, some records still have missing corresponding coordinate among our dataset, so of the map showing, we just extract the observation which has intact data.

    2) We realized the format of time in the dataset is too long to show suitbally on our map,and we also want to sort the time, so first we parse the original time format to integer which in order to sort by increasing order,and then we convert the ranked integer to the desired format. The final format that we want is 'yyyy-mm-dd'.

  • Write a short section that discusses the dataset stats, containing key points/plots from your exploratory data analysis.

    • The core idea of analysing this dataset is the collision and injury by location to understand which area has the heaviest accidental situation, and we also focus on the variation of accident during time changing,so that the key attributes is the coordinate and time
